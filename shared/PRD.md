## 1. 概述

### 1.1 產品願景

為零基礎日語學習者打造一款**純 Web 遊戲化學習平台**，通過6種核心遊戲機制，讓用戶在60天內完成 JLPT N5 所需的全部知識儲備。

### 1.2 產品目標

| 目標 | 衡量指標 | 目標值 |
|------|---------|--------|
| 用戶活躍 | DAU/MAU 比率 | ≥ 30%（行業標竿：Duolingo 37%） |
| 學習完成 | 60天課程完成率 | ≥ 25%（行業平均 8-15%） |
| 用戶留存 | 7日留存率 | ≥ 45% |
| 用戶留存 | 30日留存率 | ≥ 20% |
| 學習效果 | 模擬考 N5 通過率 | ≥ 70% |
| 每日使用 | 平均使用時長 | ≥ 12分鐘/天 |

### 1.3 目標用戶畫像

**主要用戶**：
| 屬性 | 描述 |
|------|------|
| 年齡 | 18-35歲 |
| 語言 | 中文母語 |
| 場景 | 在日生活、日語愛好、動漫文化、旅遊準備 |
| 設備 | iPad（核心）、手機、PC 瀏覽器 |
| 痛點 | 傳統教材枯燥、APP太多選不過來、缺乏完整學習路徑、難以堅持 |

**用戶故事**：
- 作為一個剛到日本的中國人，我想快速學會基礎日語，以便在便利店、車站、餐廳能基本交流
- 作為一個動漫愛好者，我想在遊戲中學日語而不是啃教科書，以便學習不再痛苦
- 作為一個碎片時間學習者，我想每天花15分鐘就能推進學習進度，以便在通勤時也能學

## 3. 遊戲世界觀

### 3.1 故事背景

> 在一個平行世界中，日語文字擁有魔力。古老的「言靈（ことだま）」守護著日本列島的和平。然而，混沌之魔「文盲（もんもう）」降臨，封印了所有文字的力量。只有收集足夠的「言靈碎片」，才能恢復文字的力量，拯救這片土地。
>
> 你，一個來自中國的冒險者，因為擁有「漢字之力」——中文的底蘊——被選召來到這個世界。你的任務：在60天內，走遍六大區域，收集800個言靈碎片（詞彙），掌握82個咒語（語法），解鎖80個封印漢字，最終在「N5試煉塔」證明你的力量。

## 4. 核心遊戲模式（6種）

### 4.1 假名消消樂（Match & Clear）

**所屬區域**：五十音島（Day 1-7），後續關卡復習  
**學習目標**：92個假名的認讀 + 平片假名對應關係

#### 4.1.1 玩法規則

**基礎模式（Day 1-4）**：
1. 4×4 網格（16張卡片），正面朝下
2. 玩家翻開兩張卡片：
   - 平假名 ↔ 羅馬音 配對 → 消除成功
   - 不匹配 → 翻回背面
3. 目標：消除所有卡片
4. 評分：⭐⭐⭐（根據翻牌次數）
   - ⭐⭐⭐：≤ 翻牌次數 = 卡片數 × 1.5
   - ⭐⭐：≤ 翻牌次數 = 卡片數 × 2.0
   - ⭐：完成即可

**進階模式（Day 5-6）**：
1. 6×4 網格（24張卡片）
2. 平假名 ↔ 片假名 配對
3. 增加限時（90秒）
4. 翻開卡片時自動播放發音

**極限模式（Day 7 BOSS關）**：
1. 8×4 網格（32張卡片）
2. 混合配對（平↔片↔羅馬音，隨機三種組合）
3. 限時60秒
4. 每10秒未配對成功，已消除的卡片隨機復活一對

#### 4.1.2 動畫規格

| 動畫 | 技術 | 時長 | 說明 |
|------|------|------|------|
| 卡片翻轉 | CSS 3D transform | 300ms | Y軸旋轉180°，正反面不同顏色 |
| 配對成功 | Lottie JSON | 500ms | 兩張卡片飛向中央合體 → 金色粒子爆炸 |
| 配對失敗 | CSS animation | 400ms | 卡片左右震動3次 + 邊框閃紅 |
| 全部消除 | Lottie JSON | 1500ms | 煙花升空 + 角色跳起慶祝 |
| 卡片復活 | CSS + JS | 600ms | 從下方升起 + 紫色光芒（極限模式） |
| 計時器緊張 | CSS pulse | 持續 | 最後10秒：數字閃紅 + 脈動放大效果 |

#### 4.1.3 音頻規格

| 音效 | 格式 | 說明 |
|------|------|------|
| 翻牌 | MP3, ≤50KB | 輕柔紙牌翻動聲 |
| 假名發音 | MP3, ≤30KB/個 | 日語母語者錄音（女聲），翻開即播 |

### 4.2 單詞 BOSS 戰（Word Battle）

**所屬區域**：單詞村（Day 8-21），貫穿後續所有區域  
**學習目標**：800個N5核心詞彙

#### 4.2.1 玩法規則

**戰鬥流程**：
```
[BOSS出場動畫] → [出題] → [玩家作答] → [判定動畫] → [下一題] → ... → [BOSS倒下/玩家失敗]
```

1. BOSS 有 HP 血條（5-10格，每關不同）
2. 每回合顯示一個日語單詞/短語
3. 下方4個選項（中文含義），玩家在8秒內選擇
4. **答對**：
   - 玩家發動攻擊，BOSS 扣1格HP
   - 連續答對觸發連擊：2連=1.5x, 3連=2x, 5連=3x（一擊扣多格）
5. **答錯/超時**：
   - BOSS 發動反擊，玩家扣1顆❤️（共3顆）
   - 連擊重置為0
6. **勝利條件**：BOSS HP 歸零
7. **失敗條件**：玩家❤️歸零

**出題策略**（SRS 間隔重複演算法）：
- 新詞首次出現：正確答案放在隨機位置，干擾項選同類別近似詞
- 答錯的詞：2關後再次出現
- 答對的詞：5關後復習出現
- 每關新詞佔60%，復習詞佔40%

### 4.3 語法跑酷（Grammar Runner）

**所屬區域**：語法城（Day 22-35）  
**學習目標**：82個語法點的句型結構 + 語序感覺

#### 4.3.1 玩法規則

**基本流程**：
1. 角色自動從左向右奔跑（恆定速度，不可控制移動）
2. 螢幕頂部顯示目標句子（中文翻譯），如：「我在學校學日語」
3. 路上浮現日語詞塊氣泡：「私は」「学校で」「日本語を」「勉強します」
4. 同時有干擾詞塊：「勉強しません」「学校に」等
5. 玩家**按順序點擊**正確詞塊（語序必須正確）
6. 全部正確 → 句子在頂部組裝完成 → TTS 朗讀 → 角色跳過終點
7. 點錯 → 詞塊爆炸 + 角色絆倒（減速2秒）

**難度遞進**：
| 等級 | 詞塊數 | 干擾詞塊 | 速度 | 對應語法 |
|------|--------|---------|------|---------|
| Lv.1 | 3個 | 1個 | 慢 | AはBです |

### 4.4 語音道場（Voice Dojo）

**所屬區域**：貫穿所有區域  
**學習目標**：發音準確度、語調、語速

#### 4.4.1 玩法規則

**訓練類型**：

| 類型 | 說明 | 出現區域 |
|------|------|---------|
| 假名跟讀 | 逐個假名跟讀 | 五十音島 |
| 單詞朗讀 | 看圖/看字說單詞 | 單詞村起 |
| 短句跟讀 | 跟讀日常短句 | 語法城起 |
| 角色扮演 | 模擬場景對話（選台詞 + 朗讀） | 會話廣場 |

**流程**：
```
[NPC示範發音] → [玩家按住錄音] → [鬆開/自動截止] → [識別+評分] → [結果展示]
```

1. NPC 角色（小狐）先示範發音（預錄音頻 or TTS）
2. 螢幕顯示目標文字（假名 + 羅馬音 + 中文）
3. 玩家按住🎤按鈕開始錄音（最長5秒）
4. 鬆開後 Web Speech API 進行識別
5. 比對識別結果 vs 目標文本，計算相似度得分

**評分算法**：
```javascript
// 主要邏輯
const recognition = new webkitSpeechRecognition()
recognition.lang = 'ja-JP'
recognition.interimResults = false

// 評分 = 文本相似度（Levenshtein距離的反向百分比）
function score(recognized, target) {
  const distance = levenshtein(recognized, target)
  const maxLen = Math.max(recognized.length, target.length)
  return Math.round((1 - distance / maxLen) * 100)
}

// 星級
// 90-100: ⭐⭐⭐ 完美
// 70-89:  ⭐⭐ 良好
// 60-69:  ⭐ 及格
// 0-59:   ❌ 重來
```

**角色扮演模式（會話廣場）**：
1. 進入場景（如餐廳）
2. NPC 說一句日語（帶字幕）：「いらっしゃいませ！何名様ですか？」
3. 螢幕顯示2-3個回答選項：
   - A.「二人です」
   - B.「メニューをください」
   - C.「すみません」
4. 玩家選擇正確答案 → 朗讀該句 → 評分
5. 對話持續3-5個回合

```sql
  user_id BIGINT NOT NULL,
  word_id INT NOT NULL,
  correct_count INT DEFAULT 0,
  wrong_count INT DEFAULT 0,
  last_review DATETIME,
  next_review DATETIME,          -- SRS 下次復習時間
  mastery_level INT DEFAULT 0,   -- 0=新詞 1=認識 2=熟悉 3=掌握
  UNIQUE KEY uk_user_word (user_id, word_id)
);

-- 語法學習記錄
CREATE TABLE user_grammar (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  grammar_id INT NOT NULL,
  mastery_level INT DEFAULT 0,
  practice_count INT DEFAULT 0,
  last_practice DATETIME,
  UNIQUE KEY uk_user_grammar (user_id, grammar_id)
);

-- 成就表
CREATE TABLE user_achievement (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  achievement_id VARCHAR(50) NOT NULL,
  unlocked_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uk_user_achievement (user_id, achievement_id)
);

-- 模擬考試成績
CREATE TABLE user_exam_result (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  exam_set_id INT NOT NULL,      -- 1-3套
  vocabulary_score INT,
  grammar_reading_score INT,
  listening_score INT,
  total_score INT,
  pass BOOLEAN,
  details JSON,                   -- 每題答題詳情
  taken_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 每日任務完成記錄
CREATE TABLE user_daily_task (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  task_date DATE NOT NULL,
  task_type VARCHAR(30) NOT NULL, -- new_words / stage / voice / review
  completed BOOLEAN DEFAULT FALSE,
  completed_at DATETIME,
  UNIQUE KEY uk_user_task (user_id, task_date, task_type)
);

-- 題庫：單詞
CREATE TABLE word (
  id INT PRIMARY KEY AUTO_INCREMENT,
  japanese VARCHAR(50) NOT NULL,   -- 日語原文
  hiragana VARCHAR(50),            -- 假名
  chinese VARCHAR(100) NOT NULL,   -- 中文含義
  category VARCHAR(30),            -- 分類：food/animal/verb/adj...
  jlpt_level VARCHAR(5) DEFAULT 'N5',
  audio_url VARCHAR(255),
  image_url VARCHAR(255)
);

-- 題庫：語法
CREATE TABLE grammar (
  id INT PRIMARY KEY AUTO_INCREMENT,
  pattern VARCHAR(100) NOT NULL,   -- 語法句型
  meaning_cn VARCHAR(200),         -- 中文含義
  explanation TEXT,                 -- 詳細說明
  category VARCHAR(30),            -- 分類
  examples JSON                    -- 例句列表
);
```

## 9. 風險評估

| 風險 | 可能性 | 影響 | 應對方案 |
|------|--------|------|---------|
| Web Speech API 兼容性差 | 中 | 高 | 降級方案：關閉語音功能，用選擇題替代；或接入第三方語音API |
| PixiJS 在低端設備卡頓 | 低 | 中 | 粒子數量分級（高/中/低畫質），iPad 8寸實測先行 |
| 語音素材製作週期長 | 高 | 中 | Phase 1-2 先用 TTS 合成；Phase 3 替換為真人錄音 |
| 課程內容教研品質 | 中 | 高 | 參考標準教材（みんなの日本語 / 標準日本語），外聘日語教師審核 |
| 用戶留存低 | 中 | 高 | 強化每日任務+連續登入獎勵+推送提醒+社交排行榜 |
| 題庫量不足 | 中 | 中 | 初期聚焦核心800詞+82語法；後續版本擴充至N4 |
